
# Halpoplayer 

A simple, lightweight Subsonic client for iOS written in SwiftUI

## Features

- Browse, search and play your music library
- Download music to play offline
- Airplay integration
- Enter 2 different server URLs for public and local access
- Carplay widget
- View your playlists
